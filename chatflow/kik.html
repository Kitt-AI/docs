Kik Bot Tutorials¶

Abstract¶

In this tutorial we will start with creating a simple Kik bot to echo your message, then we will create a bot that shows you different cat images.

A simple echo bot¶

Let’s first create a simple echo bot using ChatFlow.

1. Create a kik bot¶

First install the kik client on your mobile device.

Then go to dev.kik.com to build your bot.

Using the Kik app, touch and then click on to scan the code, which registers a bot for you. Then answer Botsworth's questions. Tap "yes" and "Create a Bot".

Give a unique name to the bot. After bot creation, click “Configuration” on top of dev.kik.com page.

You’ll have to copy/paste “Bot Name” and “API Key” to ChatFlow in the next step.

2. Create a ChatFlow application¶

Go to the applications and create a new application. Call it “kik echo bot” and put kik In and kik Out nodes as shown on the image below.

Double click the kik In node, select “Add new kikbot-controller...” and click “Edit” button. Copy and paste “kik bot name” and “API Key” and click Add. Double click the kik Out node and select the same controller.

Now deploy the application and check how it works. When you write to your bot, it answers you with the same message.

Advanced: attachments and custom keyboards¶

In this part of the tutorial we will create a bot that shows you cat imapges of different color.

1. Importing the “Colored Cats” application¶

Import the application to ChatFlow. You can either:

  • Download json of our application and import the project on the applications page, or,
  • Use the “kik - Cat Choose” sample application when clicking “CREATE NEW APP” on the applications page.

Also please download NLU json to import it as an NLU application.

2. NLU setup¶

First, go to NLU and import the NLU json you get from 1. Importing the “Colored Cats” application. Come back to ChatFlow and open the KITT.AI NLU node and paste URL you got from NLU there. Add {{{payload}}}, so your final string would look like:

http://api-nlu.kitt.ai/v1/api/900584f4-7360-479c?token=xxx?q={{{payload}}}

3. Set up “Colored Cats” application¶

Now you should register the kik bot, as described in 1. Create a kik bot section. Copy and paste bot name and bot API key, as described in 2. Create a ChatFlow application section.

After deploying your application, you can write to your bot and tell him to show you the ct of chosen color.
